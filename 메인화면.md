# 메인 화면

SK 홈페이지의 히어로 화면(전체화면 이미지 배경 + 타이핑되는 카피 + 우측 하단 click)을 참고하여 첫 화면을 구성한다.
버튼(click) 클릭 시 히어로 화면이 사라지고 5개 카테고리가 가로로 나란히 나열된 화면으로 전환된다.

> 히스토리: 네이버식 구조 -> SK커리어 스타일(히어로+버튼) -> 참고 이미지(SK 홈페이지 히어로 스크린샷) 기반 타이핑 효과/화면 전환 -> 이미지 전체화면 확대 + 상단 네비게이션 제거 + 카테고리 가로 1열 배치로 수정 -> 사이트 전체를 다크 미니멀 테마(Awwwards의 pp-neue-montreal 참고: 검정 배경 + 강렬한 레드 포인트 + 굵고 큰 타이포그래피)로 재단장.
> 색상: Claude 브랜드 웜톤 팔레트 -> 다크 테마(검정 배경) -> 화이트 배경 기반으로 최종 정리. `--bg`/`--surface`(#FFFFFF 화이트) / `--text`(#111111 블랙) / `--accent`(#E8402E 레드) 조합의 화이트×블랙×레드 미니멀 테마. 히어로 영역은 이미지 위 어두운 오버레이 + 흰 텍스트 유지(사진 가독성 목적, 전체 테마와 무관).

> 주의: index.html은 공통 네비게이션(nav)을 사용하지 않기로 함 — 카테고리 화면 자체가 다른 페이지로의 이동 수단이기 때문. frame.md의 "모든 페이지 공통 네비게이션" 체크리스트와는 다른 예외 케이스이며, 다른 5개 페이지는 공통 네비게이션을 그대로 유지함.

## 구성

1. **히어로 화면** (`section#hero-screen`)
   - 이미지(`images/welcome.jpg`)가 화면(뷰포트) 전체를 덮음 (`object-fit: cover`)
   - 이미지 위에 어두운 그라데이션 오버레이 (텍스트 가독성 확보)
   - 좌측 하단: "SKALA" 타이틀이 한 글자씩 타이핑되어 나타난 뒤, 더 작은 크기의 "환영합니다." 문구가 페이드인
   - 우측 하단: "시작하기 →" 클릭 버튼 (타이핑 완료 후 함께 페이드인)
   - 텍스트 줄 위에 얇은 가로선(구분선) 배치

2. **카테고리 화면** (`section#category-screen`)
   - 히어로 화면과 같은 위치에 겹쳐있다가, 버튼 클릭 시 히어로 화면은 페이드아웃, 카테고리 화면은 페이드인 (화면 전환)
   - 5개 카테고리를 가로로 한 줄(1행)에 나란히 배치, 순차적으로(staggered) 페이드인
   - 화면이 좁아 5개가 다 안 들어가면 가로 스크롤로 나머지 확인 (모바일 포함)
   - 항목: 내소개 / 내 강의 일정 / 회원가입 / 회원가입결과 / 내 여행지
   - 각 섹션 구성: 소제목(h3) + 한 줄 설명(p) + "바로가기" 링크(a)
   - 하단에 푸터 포함

## 구현 프롬프트 (그대로 사용 가능)

```
index.html을 아래 구조로 만들어줘. 외부 라이브러리는 쓰지 말고 순수 HTML + CSS + 최소한의 바닐라 JS만 사용해줘.
공통 네비게이션(nav)은 index.html에는 넣지 않는다 (카테고리 화면이 곧 이동 수단).

1. 히어로 화면(section#hero-screen)
   - images/welcome.jpg 이미지가 화면 전체를 덮음 (object-fit: cover)
   - 이미지 위 어두운 그라데이션 오버레이로 텍스트 가독성 확보
   - 좌측 하단에 "SKALA" 텍스트가 자바스크립트로 한 글자씩 타이핑되며 나타남
   - 타이핑 완료 후 더 작은 글씨로 "환영합니다." 문구가 페이드인
   - 우측 하단에 "시작하기 →" 버튼이 함께 페이드인, 위에 얇은 가로 구분선

2. 카테고리 화면(section#category-screen)
   - 히어로 화면과 같은 자리에 겹쳐 있다가 버튼 클릭 시
     히어로 화면 페이드아웃 + 카테고리 화면 페이드인 (화면 전환)
   - 5개 카테고리를 가로로 한 줄에 나란히 배치, 순서대로 페이드인
     - 내소개 -> myProfile.html
     - 내 강의 일정 -> myClass.html
     - 회원가입 -> signUp.html
     - 회원가입결과 -> signUpResult.html
     - 내 여행지 -> myTrip.html
   - 화면이 좁으면 가로 스크롤로 나머지 카테고리 확인
   - 각 섹션: 소제목(h3) + 한 줄 설명(p) + "바로가기" 링크(a)
   - 하단 푸터 포함
```

## 원칙
- 이미지 1장 사용 (`images/welcome.jpg`), 히어로 화면 전체를 덮음
- 색상은 Claude 브랜드 웜톤 팔레트 기본 + 히어로 영역은 이미지 대비를 위한 어두운 오버레이/흰 텍스트
- 외부 라이브러리 사용 금지 — 순수 CSS 트랜지션/애니메이션 + 최소한의 바닐라 JS(타이핑 효과, 클릭 시 화면 전환)만 사용
- 카테고리 화면은 가로 1행 배치 (그리드 wrap 아님), 좁은 화면은 가로 스크롤로 대응
- 모바일 대응: 미디어 쿼리로 히어로 텍스트 크기, 카테고리 카드 폭 등 반응형 유지
